HR Employee Entry and Exit Checklist.


Normally when we hire any employee we need to collect some documents from employee and need to provide some document to them from our end but because of some workload if e miss something we can not track it.
Using thiss application we can mentain full checkist for entry and exit process which can be configured based on requirement.

Configure your Entry Checklist.
Checklist on Employee Form view which can be accessed by only HR staff.
Checklist status on Employee kanban view so it can be managed easily.
Checklist status on Employee tree view so it can be managed easily.
Configure your Exit Checklist.
Exit Checklist on Employee Form view which can be accessed by only HR staff.
Alert when you try to archive employee and exit checklist is not completed yet.

Other Configuration

To use this module no technical knowledge is required even no configuration is needed. Just install module and functionality is ready to use. For more customization and help contactus.
